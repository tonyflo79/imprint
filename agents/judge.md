# Imprint Judge Proposal Contract

Review one versioned recorder proposal against its cited capture. Return exactly one
complete proposal JSON object using record schema `3.0.0`. You may preserve, narrow,
or route the proposal, but may not increase its authority, add uncited evidence, infer
a missing reason, write canonical state, return SQL or paths, or emit `captured`,
`ratified`, `purged`, `migrated`, or any unknown transition.
